Perkembangan Teknologi Web

Infografik ini menggambarkan evolusi teknologi web dari awal kemunculannya hingga era modern saat ini, menampilkan teknologi dan standar utama yang telah membentuk web menjadi seperti sekarang.

Era Awal (1990-1995)

1990
HTML Diciptakan
Tim Berners-Lee menciptakan HyperText Markup Language (HTML) sebagai bagian dari proyek World Wide Web di CERN.
1993
Browser Mosaic
Browser web grafis pertama, Mosaic, diluncurkan dan mempopulerkan akses web untuk masyarakat umum.

Era Dinamis (1995-2005)

1995
JavaScript & PHP
JavaScript diciptakan oleh Netscape, dan PHP dirilis oleh Rasmus Lerdorf, memungkinkan web yang lebih dinamis.
JavaScript PHP
1996
CSS
Cascading Style Sheets (CSS) diperkenalkan untuk memisahkan konten dari presentasi visual.
CSS

Era Web 2.0 (2005-2010)

2005
AJAX
Teknologi AJAX memungkinkan aplikasi web yang lebih responsif dan interaktif tanpa perlu memuat ulang halaman.
AJAX jQuery
2008
HTML5
Spesifikasi HTML5 mulai dikembangkan, membawa elemen semantik dan API canggih untuk pengalaman web yang kaya.
HTML5

Era Modern (2010-Sekarang)

2010
Framework JavaScript
Kemunculan framework seperti AngularJS, React, dan Vue.js yang mengubah cara pembuatan aplikasi web.
React Vue Angular
2015
Progressive Web Apps
Konsep PWA diperkenalkan, menggabungkan keunggulan web dan aplikasi mobile native.
PWA Service Workers
2020+
WebAssembly & AI
WebAssembly memungkinkan kode berperforma tinggi di browser, dan integrasi AI menjadi semakin umum.
WebAssembly AI/ML
